## 目录结构说明

本目录包含 iOS 版 VolcEngineRTC 的语音聊天室场景 Demo 源代码

```
├─ veRTC_Demo                   
│  │─ Feature               // 场景选择、用户信息等功能
│  
├─ Pods 
│  ├─ Development Pods  
│  │  ├─ Core               // 通用组件
│  │  ├─ Login              // 登录模块 
│  │  ├─ VoiceChatDemo      // 语音聊天室场景源码 
│  ├─ Pods                  
│  │  ├─ VolcEngineRTC      // RTC SDK

```
